# PASSION THEME

## Installation

- Open the Extensions sidebar in VS Code
- Search for Passion Theme
- Click Install and Reload
- Open the Command Palette with `Ctrl+Shift+P`
- Select Preferences: `Color Theme and choose a Passion variant`.
